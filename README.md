# MyBudget

Спрощений вебзастосунок для денного прогнозу ліквідності. Ви вводите тільки наявні кошти, денний процент оперативних витрат, процент відведення в накопичення і горизонт прогнозу.

## Як тепер працює модель
- `Наявні кошти` це стартовий ліквідний баланс.
- `Оперативні витрати, % на день` знімаються щодня від поточного доступного залишку.
- `У накопичення після витрати, %` переводить частину решти в окремий накопичувальний пул.
- Після кожної витрати база для наступного дня оновлюється.
- Історія всіх розрахунків зберігається в `storage/history.json`.

## Що показує інтерфейс
- ліквідність на фініші
- накопичені зайві кошти
- сумарний капітал
- найтісніший safe pace
- чотири сценарії
- графік ліквідності
- історію попередніх запусків

## Запуск
Якщо ви стоїте в зовнішній папці `C:\Users\grigo\Desktop\Mybuget`, найпростіші варіанти:

```powershell
python .\app.py
```

або

```powershell
.\run.bat
```

Обидва варіанти самі використовують внутрішню `.venv` з папки `.\Mybuget`.

## Ручний запуск з папки застосунку
```powershell
cd .\Mybuget
& ".\.venv\Scripts\python.exe" ".\app.py"
```
