# MyBudget

Інтерактивний калькулятор бюджету: HTML фронт + Python (Flask) бекенд.

## Що рахує
- Помісячний прогрес балансу
- Розподіл на зайві кошти і збереження у %
- Ріст операційних витрат щомісяця
- Підсумковий індекс стабільності

## Запуск
1. Створіть віртуальне середовище:
   - Windows PowerShell:
     - `python -m venv .venv`
     - `.\.venv\Scripts\Activate.ps1`
2. Встановіть залежності:
   - `pip install -r requirements.txt`
3. Запустіть застосунок:
   - `python app.py`
4. Відкрийте у браузері:
   - `http://127.0.0.1:5000`
